  2. 對網路圖示點擊右鍵，再點擊「開啟網路與網際網路設定」

  3. 在「變更您的網路設定」下，點擊「變更介面卡選項」

  4. 點擊「乙太網路」兩下，點擊底下的「內容」，再點選「 網際網路通訊協定第 4 版」兩下
    註：乙太網路也可能顯示為「有線網路」或「區域網路」

  5. 點選「使用下列的 IP 位置」與「使用下列的 DNS 伺服器位置」
    輸入剛剛查到的 IP、子網路遮罩、預設閘道與 DNS 伺服器，並點擊「確定」即可！

    若電腦本身無網路孔，請購買轉換器。

  2. 點擊左上角的蘋果圖示，點擊「系統偏好設定」，再點擊「網路」

  3. 點擊「乙太網路」，將「設定 IPv4」選為「手動」
    輸入剛剛查到的 IP、子網路遮罩、路由器，點擊「進階」
    註：若使用 USB、Thunderbolt 或 Type C 網路轉接線，請選擇相對應的選項

  4. 點擊「＋」，輸入 DNS 伺服器，再點擊「好」

  5. 點擊「套用」即可！
